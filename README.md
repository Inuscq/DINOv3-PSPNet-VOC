# DINOv3 + PSPNet 语义分割实验复现

本项目基于 DINOv3 作为 backbone，结合 PSPNet 分割头，在 Pascal VOC 2012 数据集上实现语义分割。

## 参考工作
- DINOv3: https://github.com/facebookresearch/dinov3
- PSPNet: https://arxiv.org/abs/1612.01105
- 主要代码参考: https://github.com/TimesloverJIE/voc-dinov3-pspnet

## 个人工作
- 在 Kaggle 环境中独立完成代码整合与调试
- 解决 DINOv3 权限问题，后改用 ModelScope 加载模型
- 完成完整训练流程（60 epochs），最佳 mIoU 达到 0.6599
- 生成完整的评估报告（定性对比、每类 IoU、混淆矩阵、失败案例）
  
## 实验结果
- **Backbone**: DINOv3 ViT-S/16
- **分割头**: PSPNet
- **数据集**: Pascal VOC 2012
- **最佳 mIoU**: 0.6599
- **模型权重**：‘best_pspnet.pth’
- **改进**：ASPP并联改进，最终得到mIoU为0.709（未上传模型权重）
- **改进后完整代码**：'dinov3_PPM+ASPP.ipynb'

### 5. `per_class_iou.png` — 21个类别IoU柱状图

**内容**：横轴为21个类别名称，纵轴为IoU值（0-1之间），整体 mIoU 为 0.7713。

**整体分析**：

- **高IoU类别（>0.85）**：背景、飞机、火车、公交车、人、羊等。这些类别通常具有体积大、轮廓清晰、样本充足的特点。
- **中等IoU类别（0.70-0.85）**：马、摩托车、瓶子、汽车、猫、牛、电视、船、餐桌、狗等。这些类别识别效果中等，但受形态多样性或遮挡影响。
- **低IoU类别（<0.70）**：沙发、盆栽、自行车、椅子。这些类别具有小体积、形态多变、易混淆、常被遮挡等特点，是模型的主要短板。

**结论**：
1. 模型在**大物体、轮廓清晰、样本充足**的类别上表现优秀。
2. 模型在**小物体、形态多变、易混淆、常被遮挡**的类别上表现较差。
3. 椅子（0.40）和自行车（0.51）是模型最薄弱的两个类别，需要针对性改进。
4. 整体 mIoU 0.7713 达到了较好水平，优于论文复现时的 0.6599，说明最终训练的模型效果显著提升。
  
### 6. `confusion_matrix.png` — 21×21混淆矩阵

**内容**：横轴为预测类别，纵轴为真实类别，颜色越深表示比例越高。对角线为正确分类比例。
<img width="1500" height="1200" alt="confusion_matrix" src="https://github.com/user-attachments/assets/8533da9a-e067-469e-b68d-c4d3eb9320ef" />

**主要混淆对分析**：
- **椅子 → 沙发**：约0.20。椅子有约20%的概率被误判为沙发，两者外观高度相似，是模型最大的混淆对。
- **沙发 → 椅子**：约0.15。沙发也有约15%的概率被误判为椅子，双向混淆严重。
- **自行车 → 摩托车**：约0.15。两轮交通工具轮廓相近，易混淆。
- **盆栽 → 背景**：约0.20。小物体常被模型忽略，误判为背景。
- **牛 → 羊**：约0.08。四足动物形态相似，存在一定混淆。
- **狗 → 猫**：约0.08。宠物类别间存在轻微混淆。
  
**结论**：
1. **整体分类准确**：对角线数值普遍较高（多数>0.70），说明模型对大多数类别的分类是正确的。
2. **大物体优于小物体**：飞机、火车、公交车等大物体正确分类比例高（>0.85），盆栽等小物体正确分类比例低（<0.65）。
3. **主要混淆集中在外观相似类别**：椅子↔沙发、自行车↔摩托车、猫↔狗。
4. **小物体易被忽略**：盆栽有约20%被误判为背景，说明模型对小物体的捕捉能力有待加强。
5. **改进方向**：针对混淆严重的类别对（椅子/沙发、自行车/摩托车），可考虑增加该类别的训练样本或采用更强的数据增强策略。

## CRF 后处理优化

为进一步提升分割边界的精细度，在模型输出后引入了 **Dense CRF** 作为后处理模块。CRF 利用原始图像的像素颜色和位置信息，对模型输出的粗略分割图进行边界优化。

## CBAM 注意力模块改进

为进一步提升模型的特征表达能力，在 PSPNet 的金字塔池化模块（PPM）之前引入了 **CBAM（Convolutional Block Attention Module）** 注意力机制。CBAM 通过通道注意力和空间注意力的串联结构，自适应地增强关键特征、抑制无关信息。
<img width="1525" height="538" alt="image" src="https://github.com/user-attachments/assets/7bead164-b141-49ea-ab62-29e4db551724" />


### 结果分析

- **正向提升**：CBAM 的加入使 mIoU 略有提升，验证了注意力机制在该任务上的有效性。
- **提升幅度较小的原因**：
  1. 基线模型已达到较高水平（0.6599），进一步优化空间有限
  2. PSPNet 自带的 PPM 模块已具备多尺度上下文融合能力，与 CBAM 存在一定功能重叠

## ASPP 并联 PPM 模块改进

在 CBAM 注意力机制的基础上，进一步引入了 **ASPP（空洞空间金字塔池化）** 模块，并将其与原有的 PPM（金字塔池化模块）**并联融合**，以增强模型的多尺度上下文捕捉能力。

- **PPM** 通过不同尺度的平均池化捕捉全局和区域上下文，但对小物体和细节的保留能力有限。
- **ASPP** 通过不同膨胀率的空洞卷积，在不损失分辨率的前提下扩大感受野，能更好地保留小物体和边界细节。
- 二者并联可以实现**优势互补**：PPM 提供全局场景先验，ASPP 增强局部细节和多尺度特征。

 ### 结果分析
- **显著提升**：ASPP+PPM 并联结构带来了约 5% 的 mIoU 提升（提升至0.709），是本次实验中效果最显著的改进。
